# BigMarket Contracts

## Description

A system of smart contracts, based on [Executor DAO](https://github.com/Clarity-Innovation-Lab/executor-dao),
for community ownership and management of on-chain prediction market machinery.

## Development

- Vite
- Clarinet SDK
- Stacks.js

To run tests:

```bash
git clone ..
npm install
npm run test
```
